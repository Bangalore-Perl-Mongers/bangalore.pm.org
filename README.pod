=head1 NAME

Source code of Bangalore.pm.org website

Here's the source code of Bangalore.pm.org website. A website for Bangalore Perl Mongers. It's built using mojomojo (http://mojomojo.org/)

See INSTALL or lib/MojoMojo/Installation.pod
